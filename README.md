# HeroesApp

## Dev
1. Clonar el proyecto
2. Ejecutar  ```npm install``
3. Levantar backend ```npm run backend```
4. Ejecutar la app ```npm start```o bien ```ng serve -o``  

## Heroes App - Tecnologías Utilizadas
<!-- Framework y Lenguaje -->
1. Angular 17
2. TypeScript

 <!-- Estilos y Diseño -->

2. Angular Material
3. Bootstrap 5
